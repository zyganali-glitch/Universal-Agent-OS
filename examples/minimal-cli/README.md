# Minimal CLI Example Repo

This example shows how the Agent Governance OS Starter Kit lands in a non-web repository centered on a CLI or automation tool.

Included surfaces:
- `AGENTS.md`
- `AGENT_OS_PLAN_TEMPLATE.md`
- `plans/`

Suggested demo:
1. Install one locale pack into the repo root.
2. Ask the agent to start with the consultation-first bootstrap flow.
3. Confirm that the Phase-0 interview asks about CLI usage, runtime environment, packaging, and offline/deploy expectations.
4. Ask the agent to harden the repo-root template and create the first roadmap plus child execution plan.
5. Verify that implementation stays blocked until the plan and gate strategy exist.
