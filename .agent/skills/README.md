# Root .agent/skills Index

Root `.agent/skills/` now serves two purposes:

1. locale-neutral template-library surfaces used by the package during Phase 1+
2. an index pointing to locale-pack skills and generated target-repo skills

Use:
- `en/.agent/skills/`
- `tr/.agent/skills/`
- `.agent/skills/templates/` for canonical root skill templates

The bootstrap/interviewing agent may generate more skill files and role-specific agent files as required by the current project and agent ecosystem.
